# Enumerables

The goal of this project was to duplicate some Enumerable class methods in order to learn about how to create methods inside classes.

## Output
```ruby
Input

my_numbers = [5 ,6, 7, 8]
my_numbers.my_inject {|sum, number| sum + number}

Output

> 26
```
